# PictoBlox Setup & Programming Guide

PictoBlox is a graphical block-based programming software based on Scratch 3.0, designed to make hardware programming and AI/ML concepts easy and interactive for makers.

For official installation instructions, board configuration guides, interface overviews, and step-by-step tutorials, please refer to the official STEMpedia documentation:

**[STEMpedia PictoBlox Official Documentation](https://ai.thestempedia.com/docs/pictoblox/)**

**[Programming Arduino Boards with PictoBlox](https://ai.thestempedia.com/docs/pictoblox/arduino-uno-with-pictoblox/program-arduino-board-with-pictoblox/)**
