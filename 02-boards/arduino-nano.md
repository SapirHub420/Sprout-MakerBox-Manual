# SprouT Base Board for Arduino Nano v3

The Arduino Nano v3 is a compact microcontroller board based on the ATmega328P chip. When paired with the SprouT Base Board for Arduino Nano, it provides a safe, structured, and modular plug-and-play electronics experience for makers.

## Technical Specifications

### Microcontroller Specifications
* Microcontroller: ATmega328P
* Operating Voltage: 5V
* Digital I/O Pins: 14 (6 PWM outputs)
* Analog Input Pins: 8
* Flash Memory: 32 KB
* Clock Speed: 16 MHz

---

## Base Board Custom Features

### 1. Plug-and-Play Module Ports
* Connects sensors and outputs using keyed, modular cables to prevent miswiring.
* Eliminates the need for breadboards and jumper wires.

### 2. Dedicated 5V Servo Pins
* Integrated 3-pin headers (Signal, 5V Power, Ground) for direct connection of servo motors.
* Delivers a stable 5V output supply across all servo headers.

### 3. DC Power Input Terminal Block (7-12V DC)
* Features an onboard terminal block that accepts external power inputs between 7V and 12V DC.
* Regulates incoming external voltage to a steady 5V supply for the board and connected components.

---

## Safety Guidelines

When connecting external power via the 7-12V DC terminal block, ensure correct polarity (+ and -) and do not exceed 12V DC to avoid damaging the power regulation circuitry.
