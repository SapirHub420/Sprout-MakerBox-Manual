# SprouT MakerBox Component Catalog

This manual catalog provides a complete, easy-to-reference list of all the input and output components included with the SprouT MakerBox, along with a brief description of each one.

---

## Input Components
Input components are physical controls and communication modules that detect user interaction or exchange data with external devices.

* **Bluetooth Module:** A wireless communication module that allows the SprouT board to send and receive data via Bluetooth, enabling smartphone or wireless control.
* **Button (Push Button):** A simple tactile momentary switch that sends a digital trigger signal to the controller when pressed.
* **Potentiometer:** A rotary dial knob that acts as a variable resistor, providing a precise analog input value to control variables like speed, volume, or angle.

---

## Input Components (Sensors)
These modules are designed to detect and measure physical or environmental conditions, translating them into electrical signals for the controller.

* **Air Quality Sensor:** Detects the presence and concentration of gas particles or pollutants in the surrounding air, outputting a variable analog voltage.
* **DHT11 Sensor:** A digital environmental climate sensor that measures both surrounding air temperature and relative humidity.
* **IR (Infrared) Sensor:** An optical detection sensor that emits and detects reflected infrared light, commonly used for obstacle detection or line tracking.
* **LDR (Light) Sensor:** A photoresistor-based light sensor that measures ambient brightness and converts it into a variable analog value.
* **Pressure Sensor:** A touch-sensitive resistor that changes its electrical resistance based on the mechanical force or weight applied to its surface.
* **Sound Sensor:** An acoustic module equipped with a microphone to detect surrounding environmental noise and sound triggers.
* **Tilt Sensor:** A motion-activated orientation switch containing a small rolling metallic ball that completes a circuit when the module is tilted.
* **Ultrasonic Sensor:** A non-contact distance measurement sensor that calculates range by emitting high-frequency sound waves and timing their echo reflection.
* **Vibration Sensor:** A motion detector containing a small internal spring mechanism that registers physical impacts, jolts, and vibrations.

---

## Output Components
Output components receive commands from the main controller to interact with the external world using light, sound, mechanical movement, or electrical control.

* **Buzzer:** An audio signaling component used to generate distinct beep sounds, alarms, or custom musical frequencies.
* **DC Motor:** A standard direct-current motor that translates electrical power into continuous high-speed rotational movement.
* **LCD:** A liquid crystal character screen that displays text, numeric values, and customized real-time project updates directly from your code.
* **LED:** A highly efficient light-emitting diode used for status indicators, signal flashes, and visual lighting effects.
* **Motor Driver:** A power-handling circuit board that uses safe, low-power signals from the controller to drive and manage high-current DC motors.
* **Relay:** An electromagnetic switch that allows low-voltage signals from the controller to safely control high-power and high-voltage electrical devices.
